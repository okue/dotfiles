# Cheat Sheet

[sheet](https://vim.rtorr.com/lang/ja/)


## Vim
- 'vim -p' : open with tab
- 'vim -o' : open with vsplit

### Tree
- '\e'     : open and close tree
- 't, T'   : open tab

### Comment
- '\c'     : comment out

### Tab
- '\t'     : move to next tab
- 'gt'     : move to next tab
- 'gT'     : move to prev tab

### Window
- '\w + w' : move window
- 'ctrl w' : move window

### Ale
- 'ctrl j' : next warning, error
- 'ctrl k' : prev warning, error
